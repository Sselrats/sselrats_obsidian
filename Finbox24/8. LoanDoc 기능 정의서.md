## 1. 인증 & 회원 (customers, sessions, social_accounts, verification_codes)

### User API

- 회원가입 (전화번호, 카카오)
- 로그인/로그아웃
- 휴대폰 인증번호 발송/검증
- 프로필 조회/수정

### Admin API

- 고객 목록 조회 (검색/필터링)
- 고객 상세 조회
- 고객 통합 이력 조회 (고객원장)

---

## 2. 고객 상세정보 (customer_details, tmp_customer_details)

### User API

- 고객 상세정보 입력/수정 (소득, 신용점수 등)
- 임시 정보 저장 (회원가입 전)
- 임시정보 -> 정회원 변환
- 본인인증 (KCB)

### Admin API

- 고객 상세정보 이력 조회
- 임시 정보 → 정회원 변환 조회

## Overview
input u in R^m
phi(u) = sigma(w^T u + b)
w in R^m weight vector. b in R bias term
sigma R -> R nonlinear function
w와 b 가 학습을 통해 찾아내고자 하는 parameter

single one -> extend!
layer consists of multiple neurons
h = sigma (W u + b)
W in R^(k * n), b in R^k, sigma elementwise
## feed forward NN
x in R^d, L depth
h^(0) = x, h^(l) - sigma (W^(l) h^(l-1) + b^(l)), l = 1 ~ L-1
then f_THETA (x) = W^(L)h^(L-1) + b^(L) where THETA = { W^(l) , b^(l), l = 1 to L }
L : composition의 개수
nonlinear 가 아니라 linear 면 재미가 없다
